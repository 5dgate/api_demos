5DGate API demos.
========

本项目是黑曜石公司 API 接口对接的示例代码。


项目目录说明
========


```
api_demos           项目根目录
    java            Java 示例
    python          Python 示例
    REAMDME.md      说明文档
```

请根据贵司的项目需要，参考相应语言的示例代码。

请仔细阅读，各目录下的 README.md 说明文档。

各编程语言目录下的示例代码，均通过了我司专业的技术人员的测试。
